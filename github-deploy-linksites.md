# 🚀 GitHub Pages Deploy - linksites.github.io/democrata

## Configuração Personalizada

### 1. **Criar Repositório**
```
1. Acesse: https://github.com/linksites
2. Clique em "New repository"
3. Repository name: democrata
4. Description: Site Democrata Barbearia & Pub
5. Public
6. NÃO adicione README, .gitignore ou license
7. Clique em "Create repository"
```

### 2. **Comandos Git**
```bash
cd c:\Projeto\democrata
git init
git add .
git commit -m "🚀 First commit - Democrata Barbearia & Pub

✨ Features:
- Modern responsive design
- WhatsApp, iFood, Instagram integration
- Interactive gallery with lightbox
- Floating action buttons
- Performance optimized

🎯 Ready for: https://linksites.github.io/democrata"

git remote add origin https://github.com/linksites/democrata.git
git branch -M main
git push -u origin main
```

### 3. **Ativar GitHub Pages**
```
1. Vá para: https://github.com/linksites/democrata
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: main
5. Folder: / (root)
6. Save
```

### 4. **URL Final**
```
🎯 https://linksites.github.io/democrata
```
