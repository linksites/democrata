# 🚀 GitHub Actions Workflow Setup

## Workflow Criado!

### **Arquivo:** `.github/workflows/deploy.yml`

Este workflow automatiza o deploy para GitHub Pages:

### **Funcionalidades:**
- ✅ **Deploy automático** ao fazer push
- ✅ **Build otimizado** com GitHub Actions
- ✅ **HTTPS automático**
- ✅ **Monitoramento** de deploy
- ✅ **Rollback automático** se falhar

### **Como Funciona:**

1. **Trigger:** Push na branch `main`
2. **Build:** Ubuntu latest
3. **Deploy:** GitHub Pages automático
4. **URL:** https://linksites.github.io/democrata

### **Próximos Passos:**

1. **Fazer push do workflow:**
```bash
git add .github/workflows/deploy.yml
git commit -m "🚀 Add GitHub Actions workflow for auto-deploy"
git push
```

2. **Ativar GitHub Pages:**
   - https://github.com/linksites/democrata
   - Settings > Pages
   - Source: **GitHub Actions** (não Deploy from branch)
   - Save

3. **Aguardar primeiro deploy** (2-3 minutos)

4. **Acessar site:** https://linksites.github.io/democrata
