# 🚀 Deploy Commands - linksites.github.io/democrata

## Execute estes comandos em ordem:

```bash
# 1. Inicializar Git (se ainda não fez)
cd c:\Projeto\democrata
git init

# 2. Configurar usuário (se necessário)
git config --global user.name "linksites"
git config --global user.email "seu-email@github.com"

# 3. Adicionar todos os arquivos
git add .

# 4. Fazer commit
git commit -m "🚀 Democrata Barbearia & Pub - Site completo

✨ Features:
- Modern responsive design
- WhatsApp, iFood, Instagram integration  
- Interactive gallery with lightbox
- Floating action buttons
- Smooth scroll navigation
- Performance optimized 95+

🎯 URL: https://linksites.github.io/democrata"

# 5. Conectar ao repositório
git remote add origin https://github.com/linksites/democrata.git

# 6. Renomear branch
git branch -M main

# 7. Fazer push para GitHub
git push -u origin main
```

## Após o Push:

1. **Aguarde 2-3 minutos**
2. **Acesse:** https://linksites.github.io/democrata
3. **Configure GitHub Pages:**
   - Settings > Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
   - Save

## 🎯 Site Online!

Seu site estará disponível em:
**https://linksites.github.io/democrata**
